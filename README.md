# Puffin-OpenGL-Engine

## Features
  - Fog
  - Dynamic shadow mapping (directional and point lights) with PCF
  - Water reflection and refraction
  - Normal mapping
  - Mesh outline using stencil buffer
  - Particle effects
  - Postprocessing
  - TrueType font rendering, text outline
  - Antialiasing
  - Skybox reflections
